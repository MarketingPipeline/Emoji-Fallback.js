How to add Emoji Fallback support on your website on your website

Add this <script> Emoji-Fallback.js to the<bottom> of your HTML document.

<script type="module" src="https://cdn.jsdelivr.net/gh/MarketingPipeline/Emoji-Fallback.js/version/1.0.0/dist/emoji-fallback.min.js"><script>

To change / set - fallback Emoji sizes add this CSS to your website with your prefered height / width for emojis. 


img.emoji {
  display: inline !important;
  border: none !important;
  box-shadow: none !important;
  height: 1em !important;
  width: 1em !important;
  margin: 0 .07em !important;
  vertical-align: -0.1em !important;
  background: none !important;
  padding: 0 !important;
}

You're done! Your website now has Emoji support on all devices!
